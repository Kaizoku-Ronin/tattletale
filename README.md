# 📋 Tattletale

*For the record.*

A dead-simple incident log that runs entirely in your browser. Type a name, set a date and time (or hit **Now**), write down what happened, and it's on the record. When you've got enough, print a clean, formal log sheet.

Built for anyone who's tired of keeping receipts on pen and paper.

*Available at*
 
 **https://kaizoku-ronin.github.io/tattletale/**


## Features

- **One-tap timestamps** — the "Now" button stamps the current date and time
- **Name autocomplete** — names you've logged before fill in automatically
- **Filter by person** — view (and print) the record for one name at a time
- **Running tally** — see the incident count at a glance
- **Printable sheet** — the on-screen notebook prints as a sober, professional incident table with date, time, name, and description
- **Saves automatically** — entries persist in your browser between visits
- **No accounts, no server, no tracking** — everything stays on your device

## Usage

Open the site, log an entry, done. Bookmark it on your phone for quick access.

Tips for a useful record:

- Log things as they happen — contemporaneous notes carry more weight
- Stick to facts: what happened, when, for how long
- Skip the editorializing; the dates and times speak for themselves

## Privacy

Tattletale has no backend. Entries are stored locally in your own browser (localStorage) and never leave your device. Clearing your browser data will erase the record — print or screenshot anything you want to keep.

## Running it yourself

It's a single HTML file. Clone the repo and open `tattletale.html` in any browser, or host it anywhere static files are served (GitHub Pages works great).

```
git clone https://github.com/Kaizoku-Ronin/tattletale.git
```
